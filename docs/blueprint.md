# **App Name**: Bilacert Admin Pro

## Core Features:

- Authentication: Secure login for administrators using Email/Password through Firebase.
- Real-time Dashboard: Display real-time metrics from Firestore collections and Supabase using custom hooks to reflect new submissions instantly.
- Submission Pipeline: Manage submissions with a table featuring search, filter, and pagination. Include a modal for detailed views and status updates.
- Auth Guard: Protect admin routes to ensure only authenticated users can access admin panel.
- Regulatory Service Management: CRUD interface for regulatory service metadata.
- Testimonial Management: Interface for managing social proof hosted on Supabase, ensuring real-time synchronization of testimonial data.

## Style Guidelines:

- Primary color: Bilacert theme blue (#0A2B4C) for a professional and trustworthy feel.
- Background color: Light grayish-blue (#F0F4F8) for a clean and modern interface.
- Accent color: Orange (#F2994A) for interactive elements and key actions.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, machined, objective, neutral look suitable for both headlines and body text.
- Consistent and professional icons for navigation and actions.
- Mobile-first, responsive design using Tailwind CSS utility classes.
- Subtle transitions and feedback animations for user interactions.
